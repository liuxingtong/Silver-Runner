主轴均为玫红实线（无顶点圆点）。商业 / 体育认知轴：轴名从最左端点向西引出，卡片整体落在研究区外；社会认知轴（V1 / V2 / V3）：每条均从最右端点向东引出至区外。新增副轴 为玫红虚线，无轴名标签
用地数据加载后，在研究区内显示全部OSM 用地填色（按标签类型配色）；POI 不参与地图打点，仍参与资源区 / 潜力区统计。
每个多边形在地图质心处有常驻卡片，可折叠；轴线与两区会自动保存在本浏览器。关联文件 / 另存关联（Chrome、Edge）可将草稿每次编辑后自动写入你选定的本地 JSON（建议指向仓库里的 data/site-selection-draft.json），再自行 git 提交推送；亦可使用导出 JSON。GitHub Pages 在无本地草稿时会尝试加载仓库中的该文件。
